Xtreme Vulnerable Web Application (XVWA) - Instrucciones

XVWA es fácil de instalar. Puede ser configurado en Windows, Linux o Mac. Los siguientes son los pasos básicos que debes seguir en tu entorno Para la instalacion. Puedes usar WAMP, XAMP o cualquier cosa Apache-PHP-MYSQL para que funcione correctamente

Copie la carpeta xvwa en su directorio web. Asegúrese de que el nombre del directorio sea xvwa. Realice los cambios necesarios en xvwa/config.php para la conexión a la base de datos. Ejemplo a continuación:

$XVWA_WEBROOT = '';
$host = "localhost";
$dbname = 'xvwa';
$user = 'root';
$pass = 'root';

Realice los siguientes cambios en el archivo de configuración de PHP.

file_uploads = on
allow_url_fopen = on
allow_url_include = on

Acceso a la Web : http://localhost/xvwa/
Configure la base de datos y las tablas accediendo a http://localhost/xvwa/setup/

Detalles del Acceso:

admin:admin
xvwa:xvwa
user:vulnerable

Aclaración
No aloje esta aplicación en linea ni en el entorno de producción. XVWA es una aplicación totalmente vulnerable y el acceso en línea de esta aplicación podría llevar a un completo compromiso de su sistema. No somos responsables de dichos incidentes. Mantenerse a salvo por favor !
